# max-aa-exporter

This exports max pacher as ascii art.

## prerequisite
Install [graph-easy](https://github.com/ironcamel/Graph-Easy).
```
$ cpan Graph::Easy  # mac
```

## usage
```
$ poetry run python -m max_aa_exporter -f ./testdata/karplus_strong.maxpat | graph-easy
```
